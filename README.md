# SUDOKU-Agenda

**SUDOKU-Agenda** es una aplicación web diseñada para facilitar la programación de citas en nuestra academia de reforzamiento académico. Permite a los usuarios visualizar la disponibilidad de horarios y reservar clases según sus necesidades. Toda la información registrada se almacena automáticamente en una base de datos MongoDB alojada en la nube, lo que garantiza un control centralizado, seguro y eficiente.

La aplicación utiliza tecnologías modernas como **Streamlit** para la interfaz de usuario y **Python** para la lógica de backend. MongoDB, como base de datos en la nube, permite un manejo robusto de los datos, asegurando que las citas queden registradas de manera confiable y accesible.

## Características

- **Agendamiento de citas:** Los usuarios pueden consultar la disponibilidad y reservar horarios para las clases de reforzamiento.
- **Gestión en la nube con MongoDB:** Toda la información se almacena directamente en una base de datos MongoDB en la nube para garantizar un control eficiente.

## Tecnologías Utilizadas

- **Frontend y Backend:** Streamlit, Python
- **Base de Datos:** MongoDB (gestión mediante `pymongo`)

## Requisitos Previos

- **Python 3.x:** Asegúrate de tener una versión reciente instalada.
- **MongoDB Atlas:** Configura una cuenta en MongoDB Atlas para obtener la URI de conexión a la base de datos.
- **Pip:** Gestor de paquetes de Python para instalar dependencias.

## Librerías Utilizadas
- **streamlit:** Para crear la interfaz interactiva.
- **pymongo:** Para la conexión y gestión de datos en MongoDB.
- **streamlit_option_menu:** Para el manejo del menú en la interfaz.
